## 5.1 Classification de King-Moe (classification historique)

🤔 **Question de réflexion** : Pourquoi remplacer une classification qui fonctionne ? La classification de King-Moe, publiée en 1983, était le standard pendant 15 ans. Qu'est-ce qui la rendait insuffisante ?

### 5.1.1 Description des types

La classification de King-Moe (King et al., 1983, *J Bone Joint Surg Am*) définit **5 types** de scolioses idiopathiques de l'adolescent, basée sur la radiographie de face et les bending films :

> [MEDIA: 📊 M05-S01-001 — Schéma comparatif King-Moe Types I-V avec radiographies représentatives]

| Type | Description | Courbure à fusionner |
|------|-------------|---------------------|
| **I** | Courbure en S : thoracique < lombaire ; les deux courbures croisent la ligne médiane | Thoracique + lombaire |
| **II** | Courbure en S : thoracique > lombaire ; les deux croisent la ligne médiane | Thoracique seule |
| **III** | Courbure thoracique pure ; lombaire ne croise pas la ligne médiane | Thoracique seule |
| **IV** | Courbure thoracique longue fusant dans L4 (L4 inclinée vers la courbure thoracique) | Thoracique long |
| **V** | Courbure thoracique double ; T1 inclinée vers la courbure thoracique haute | Double thoracique |

### 5.1.2 Limites et raisons du remplacement

- **Pas de dimension sagittale** : King-Moe ignore entièrement le profil sagittal
- **Pas de dimension axiale** : la rotation vertébrale n'est pas intégrée
- **Ambiguïté** : les types II et III sont difficiles à distinguer en pratique
- **Pas de type thoracolombaire/lombaire** principal isolé
- **Mauvaise reproductibilité inter-observateur** (kappa = 0,61)
- La classification de Lenke (2001) a supplanté King-Moe → pratiquement **abandonné** après 2005

---

## 5.2 Classification de Lenke (2001) — Standard actuel

La classification de **Lawrence Lenke** (Lenke et al., 2001, *J Bone Joint Surg Am*) est le **gold standard** actuel pour les scolioses idiopathiques de l'adolescent. Elle est tridimensionnelle, reproductible, et directement liée à la stratégie de fusion.

> [MEDIA: 📊 M05-S02-001 — Infographie complète classification de Lenke : 6 types + modificateur lombaire + modificateur sagittal]

### 5.2.1 Principes fondamentaux

La classification Lenke repose sur **3 composantes** :
1. **Type de courbure (1-6)** : basé sur les régions thoracique proximale, thoracique principale, et thoracolombaire/lombaire
2. **Modificateur lombaire (A, B, C)** : position de la courbure lombaire par rapport à la CSVL
3. **Modificateur sagittal (−, N, +)** : cyphose thoracique T5-T12

Pour attribuer le type, on utilise les **radiographies de bending** (inclinaison latérale gauche et droite) pour déterminer si chaque région est **structurale** ou **non structurale**.

**Critères de structuralité** (courbure = structurale si l'un des critères est rempli) :
- Cobb ≥ 25° sur le bending film controlatéral
- Cyphose thoracique ≥ 20° (T2-T5 ou T10-L2)

### 5.2.2 Les 6 types de courbures Lenke

> [MEDIA: 🧊 M05-S02-002 — Modèle 3D BioDigital : 6 types Lenke interactifs (visualisation 3D de chaque type)]

| Type | Nom | TP | TM | TL/L | Fusionner |
|------|-----|----|----|------|-----------|
| **1** | Main thoracic | Non struct | **Struct** | Non struct | TM seul |
| **2** | Double thoracique | **Struct** | **Struct** | Non struct | TP + TM |
| **3** | Double majeur | Non struct | **Struct** | **Struct** | TM + TL/L |
| **4** | Triple majeur | **Struct** | **Struct** | **Struct** | TP + TM + TL/L |
| **5** | Thoracolombaire/lombaire | Non struct | Non struct | **Struct** | TL/L seul |
| **6** | Thoracolombaire/lombaire – thoracique principale | Non struct | **Struct** (minor) | **Struct** | TM + TL/L |

*TP = Thoracique proximale (T1-T5) ; TM = Thoracique principale (T6-T11) ; TL/L = Thoracolombaire/lombaire (T12-L4)*

**Type 1 (Main thoracic)** — le plus fréquent (50-55% des SIA ; Lenke et al., 2001) :
- Scoliose thoracique droite principale
- Lombaire non structurale → **ne fusionner que le thoracique**
- Exemple de Sophie : type **1**

**Type 2 (Double thoracique)** :
- La région thoracique proximale (T1-T4) est également structurale — typiquement une courbure thoracique haute gauche
- T1 est incliné vers la gauche sur la radiographie de face
- Si on ne fusionne pas la région thoracique haute → déséquilibre des épaules post-opératoire

**Type 3 (Double majeur)** :
- Les deux courbures (TM + TL/L) sont majeures → **fusion longue** obligatoire
- Difficulté : préserver autant de niveaux mobiles que possible tout en couvrant toutes les courbures structurales

**Type 4 (Triple majeur)** :
- Forme la plus englobante
- Rare (< 5%)
- Fait souvent appel à une fusion jusqu'à L3-L4

**Type 5 (Thoracolombaire/lombaire principale)** :
- Courbure lombaire gauche prédominante — moins fréquent (10-15%)
- La courbure thoracique est compensatoire non structurale → ne pas la fusionner
- Résultat esthétique important : corriger la gibbosité lombaire sans artifice thoracique

**Type 6 (TL/L-TM)** :
- La courbure TL/L est la principale mais la TM est également structurale (Cobb proche ou supérieur à la TL/L)
- Limite avec le type 3 : la règle de base est que si Cobb TL/L > Cobb TM → type 6 ; si Cobb TM > Cobb TL/L → type 3

### 5.2.3 Modificateur lombaire (A, B, C)

Le modificateur lombaire évalue la position de la **partie apicale de la courbure lombaire** par rapport à la **Ligne Centrale Vertébrale Sacrée (CSVL)** — ligne verticale passant par le milieu du sacrum.

> [MEDIA: 📐 M05-S02-003 — Schéma CSVL et modificateurs A/B/C avec exemples radiographiques annotés]

| Modificateur | Définition | Implication |
|-------------|-----------|-------------|
| **A** | La CSVL passe entre les deux pédicules de l'apex lombaire | Lombaire bien centré → sélection courte (ne pas inclure le lombaire dans la fusion) |
| **B** | La CSVL touche le bord du corps vertébral de l'apex lombaire | Lombaire modérément décentré → frontière → discuter l'inclusion |
| **C** | La CSVL est complètement latérale par rapport à l'apex lombaire | Lombaire franchement décentré → risque de décompensation si non fusionné |

**Règle pratique** :
- Modificateur **A** ou **B** → la courbure lombaire est peu problématique ; on peut souvent ne fusionner que le thoracique
- Modificateur **C** → la courbure lombaire est significative ; risque de progression lombaire post-opératoire si non incluse dans la fusion

### 5.2.4 Modificateur sagittal thoracique (−, N, +)

Le modificateur sagittal évalue la cyphose thoracique **T5-T12** :

| Modificateur | Cyphose T5-T12 | Signification |
|-------------|---------------|--------------|
| **−** (hypo) | < 10° | Hypocyphose thoracique (typique de la SIA thoracique) |
| **N** (normal) | 10-40° | Cyphose normale |
| **+** (hyper) | > 40° | Hypercyphose thoracique |

**Importance chirurgicale** :
- SIA avec hypercyphose (+) : peut nécessiter des manœuvres spécifiques de correction sagittale (pré-cintrage de la tige, compression antérieure)
- SIA avec hypocyphose (−) : la correction par dérotation restaure souvent spontanément une cyphose normale

### 5.2.5 Application pratique — Les 42 types possibles

**Formule** : Type (1-6) + Modificateur lombaire (A, B, C) + Modificateur sagittal (−, N, +)

**Exemples courants** :

| Code | Interprétation | Patient typique |
|------|--------------|----------------|
| **1A−** | Main thoracique, lombaire centré, hypocyphose | Sophie (type le plus fréquent de SIA) |
| **1B−** | Main thoracique, lombaire tangent, hypocyphose | Forme intermédiaire |
| **1CN** | Main thoracique, lombaire décentré, cyphose normale | Risque de décompensation → fusion plus longue |
| **2AN** | Double thoracique, lombaire centré, cyphose normale | Fusion jusqu'en thoracique haute |
| **3CN** | Double majeur, lombaire décentré, cyphose normale | Fusion longue T-L obligatoire |
| **5CN** | TL/L principale, lombaire décentré, cyphose normale | Fusion lombaire gauche |

**Comment le type Lenke oriente la fusion** — Règles de base :

1. **Seules les courbures structurales** doivent être incluses dans la fusion
2. La **vertèbre limitante inférieure** doit être dans la zone stable, horizontale (< 5°)
3. Le niveau de fusion proximal doit être neutre (pas d'inclinaison T1-T4)
4. Pour les types 1A−/1B− : fusion typique **T4 ou T5 à T11 ou T12**

> [MEDIA: 📊 M05-S02-004 — Tableau de sélection des niveaux de fusion selon le type Lenke (recommandations SRS)]

### 5.2.6 Reproductibilité de Lenke

- Kappa pour le type de courbure : **0.92** (excellent)
- Kappa pour le modificateur lombaire : **0.87** (excellent)
- Kappa pour le modificateur sagittal : **0.72** (bon)
*(Lenke et al., 2001, J Bone Joint Surg Am ; Niemeyer et al., 2006, Spine)*
- Bien supérieur à King-Moe (kappa = 0,61 ; Cummings et al., 1998, *J Bone Joint Surg Am*)

## 5.4 Classification SRS-Schwab — Scoliose de l'adulte

La classification **SRS-Schwab** (Schwab et al., 2012, *Spine*) a été développée par Frank Schwab et la Scoliosis Research Society pour les scolioses dégénératives de l'adulte. Elle integre les paramètres sagittaux dont on sait qu'ils déterminent la qualité de vie.

> [MEDIA: 📊 M05-S04-001 — Infographie SRS-Schwab : types de courbures + modificateurs sagittaux]

### Types de courbures (6 types)

| Type | Description |
|------|-------------|
| **T** | Thoracique seule (courbure ≥ 30°) |
| **L** | Thoracolombaire/lombaire principale (≥ 30°) |
| **D** | Double majeur (≥ 30° x 2) |
| **N** | Pas de courbure ≥ 30° (déséquilibre sagittal "pur") |

### Modificateurs sagittaux (3 modificateurs)

Trois paramètres sagittaux sont évalués, chacun gradé 0/+/++ :

| Modificateur | Paramètre | 0 (Normal) | + (Modéré) | ++ (Sévère) |
|-------------|-----------|------------|-----------|------------|
| **PI-LL** | Incidence pelvienne – lordose lombaire mismatch | < 10° | 10-20° | > 20° |
| **SVA** | Sagittal Vertical Axis (déséquilibre sagittal global) | < 4 cm | 4-9,5 cm | ≥ 9,5 cm |
| **PT** | Pente pelvienne (retroversion du bassin) | < 20° | 20-30° | ≥ 30° |

*Seuils validés par Schwab et al., 2012, Spine*

**PI-LL mismatch** — paramètre clé :
> PI-LL mismatch = Incidence Pelvienne (PI) – Lordose Lombaire (LL)
> Normal : PI-LL < 10°
> Si PI-LL > 10° → insuffisance de lordose → compensation par retroversion du bassin (↑PT) et flexion des genoux → douleur, fatigue, mauvaise qualité de vie

**Exemple** : Patient adulte avec PI = 55°, LL = 30° → PI-LL = 25° → modificateur PI-LL ++ → objectif chirurgical : restaurer LL à 45-55° (PI-LL < 10°)

### Corrélation avec les PROMs (Patient Reported Outcomes)

Les 3 modificateurs sagittaux corrèlent fortement avec le score SRS-22r et l'ODI (Oswestry Disability Index) :
- **SVA > 9,5 cm** : corrèle avec une incapacité sévère (ODI > 40)
- **PI-LL > 20°** : corrèle avec douleur lombaire et claudication importantes
- La correction des modificateurs sagittaux améliore la qualité de vie **indépendamment de la correction coronale**

## 5.5 Classifications tridimensionnelles (3D)

🤔 **Question de réflexion** : Si la radiographie standard sous-estime la vraie déformation 3D, comment classer précisément une scoliose dans tous ses plans ?

> [MEDIA: 🎬 M05-S05-001 — Animation classification 3D : mise en évidence de la déformation dans les 3 plans (frontal+sagittal+axial)]

### 5.5.1 Classification 3D de Lenke (en développement)

Extension du Lenke 2D intégrant :
- L'angle de déformation maximale de Stagnara (plan de déformation maximale)
- La rotation vertébrale apicale (AVR mesurée en EOS 3D)
- La déformation sagittale régionale

Non encore standardisée universellement — utilisée dans les centres de recherche dotés d'EOS.

### 5.5.2 Classification de Rigo (pour le traitement par corset)

La classification de **Manuel Rigo** (Rigo et al., 2010, *Scoliosis* ; SOSORT) évalue la scoliose en **vue axiale** pour adapter la construction du corset Chêneau ou Rigo-Chêneau.

> [MEDIA: 🧊 M05-S05-002 — Modèle 3D interactif classification Rigo A1-E2 avec formes de corset correspondantes]

**Logique** : Le corset agit dans les 3 plans. La position des côtes et la forme du tronc en vue axiale détermine les zones d'appui et les "fenêtres" du corset.

**Types principaux** :

| Type Rigo | Caractéristiques | Forme du tronc en axial |
|-----------|-----------------|------------------------|
| **A1** | Thoracique droite, lombaire non structurale — équilibre coronal bon | Asymétrie thoracique simple |
| **A2** | Thoracique droite avec lombaire légère | Double convexité modérée |
| **A3** | Thoracolombaire ou lombaire principale | Rotation lombaire prédominante |
| **B1** | Double courbure (thoracique + lombaire) symétrique | Deux gibbosités |
| **B2** | Double courbure asymétrique | Gibbosité thoracique > lombaire |
| **C** | Courbure thoracique gauche (atypique) | Rare — vérifier pathologie sous-jacente |
| **E1/E2** | Formes complexes avec composante cervicale | Instrumentation haute |

**Application en CFAO** (Conception et Fabrication Assistée par Ordinateur) : la classification Rigo est utilisée par les orthoprothésistes pour la fabrication numérique du corset.

### 5.5.3 Classification SRS 3D (projet en cours)

La Scoliosis Research Society travaille à une classification 3D standardisée basée sur l'imagerie EOS sterEOS, intégrant :
- Le plan de déformation maximale
- La lordose thoracique (souvent oubliée)
- La rotation axiale globale
- Les paramètres sagittaux régionaux

**Futur de la classification** : L'intelligence artificielle (deep learning appliqué aux reconstructions EOS 3D) permettra probablement une classification automatique reproductible dans les années à venir.

## 5.7 Application pratique — Classer Sophie

**Données de Sophie** :
- Radiographie de face : courbure thoracique droite T5-T12, Cobb = 28°, apex T9
- Bending droit (inclinaison gauche) : thoracique se réduit à 15° → **non structurale selon Lenke (< 25°)**
- Bending gauche (inclinaison droite) : thoracique reste à 22°
- Région thoracique proximale (T1-T4) : Cobb 8°, pas de structuralité
- Région thoracolombaire/lombaire : Cobb 18°, se réduit à 8° au bending → **non structurale**
- CSVL passe entre les pédicules de L2 (apex lombaire) → **modificateur A**
- Cyphose T5-T12 = 18° → entre 10-40° → **modificateur N**

**Classification** : **1AN**
- Type **1** : thoracique principale seule, les autres non structurales
- Modificateur **A** : lombaire bien centré
- Modificateur **N** : cyphose normale

**Implications chirurgicales** (si corset échoue) :
- Fusion thoracique seule : T4 ou T5 jusqu'à T11 ou T12
- Pas de descente en lombaire (modificateur A)
- Tige cyphosante en thoracique pour maintenir le profil (modificateur N = cyphose à préserver)
- Correction cible : Cobb post-op < 25° (60-70% de correction)
