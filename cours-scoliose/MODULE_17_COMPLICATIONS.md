## 17.1 Complications peropératoires

### 17.1.1 Déficit neurologique peropératoire

| Cause | Mécanisme | Prévention |
|-------|----------|-----------|
| **Brèche pédiculaire** | Vis en dehors du pédicule | Triggered EMG + navigation |
| **Manipulation médullaire** | Mauvaise manœuvre, traction | Monitoring PES/PEM, décision douce |
| **Ischémie médullaire** | Ligature des artères segmentaires | Éviter ligature bilateral, neuromonitoring |
| **Distraction excessive** | Correction trop rapide | Correction progressive |

**Alerte PES/PEM peropératoire** (voir Module 8) :
- Arrêt immédiat → causes systémiques → desserrage → wake-up test.

### 17.1.2 Brèche durale

- **Prévalence** : 3–8 % (chirurgies primaires) ; 10–15 % (révisions)
- **Diagostic** : Écoulement aqueux clair, hernie de sac dural
- **Traitement** :
  - Suture directe si possible (fil non résorbable 4/0 étanche)
  - Patch de colle biologique + fat graft si inaccessible
  - Fermeture étanche obligatoire (pression de confirmation)
  - Décubitus dorsal post-op 24–48h

### 17.1.3 Lésion vasculaire

- **Voie antérieure** : artère ou veine iliaque — urgence vasculaire (0,1 %)
- **Voie postérieure** : artère ou veine segmentaire — saignement contrôlable habituellement

## 17.2 Complications postopératoires précoces (J0–J30)

### 17.2.1 Hématome épidural post-opératoire

> **Urgence chirurgicale absolue**

- **Incidence** : 0,5–1 %
- **Tableau** : Parésie ou paralysie progressive des membres inférieurs dans les 6–24h post-op (pic J1)
- **Diagnostic** : Clinique + IRM urgente (ou reprise bloc si délai > 8h)
- **Traitement** : Reprise chirurgicale en urgence dans les 6–8h (délai = pronostic)

### 17.2.2 Infection du site opératoire

| Type | Timing | Profondeur | Fréquence |
|------|--------|-----------|----------|
| **Superficielle** | J0–J30 | Jusqu'à fascia | 2–3 % |
| **Profonde** | J0–J90 | Profonde / sur matériel | 1–3 % (SIA) ; 5–10 % (NM) |
| **Tardive** | > J90 | Biofilm sur implants | 1–2 % |

**Germes** :
- S. aureus SASM/MRSA : 40–60 % (Pull ter Gunne et al., 2010, *Spine*)
- Staphylococcus epidermidis (biofilm) : 20–30 % (Pull ter Gunne et al., 2010, *Spine*)
- Bacilles Gram négatifs (surtout neuromusculaire) : 10–15 %

**Facteurs de risque** :
- IMC élevé, diabète, dénutrition, corticothérapie, durée opératoire > 4h
- Profil neuromusculaire, Duchenne, SMA

**Prise en charge infection profonde précoce (< 3 mois)** :
1. Débridement-lavage-maintien du matériel (DLMM) — si < 3 mois et fusion non démontrée
2. Cultures peropératoires multiples (× 5)
3. Antibiothérapie ciblée prolongée (6–12 semaines IV puis oral)
4. Cicatrisation par seconde intention ou lambeau (plasticien)
5. Si matériel impossible à maintenir → ablation + antibio + remise de matériel différée

**Prise en charge infection tardive (> 3 mois)** :
- Ablation du matériel si fusion solide + antibiothérapie prolongée

**Prévention** :
- Antibioprophylaxie (céfazoline 2g IV dans l'heure avant incision)
- Vancomycine locale topique (powder) sur la plage osseuse (réduction iso profonde documentée)
- Préparation cutanée chlorhexidine
- Minimiser la durée opératoire
- Glycémie < 11 mmol/L peropératoire

### 17.2.3 Syndrome de queue de cheval ou déficit radiculaire

- Hématome épidural (traité ci-dessus)
- Vis mal positionnée compromettant une racine → reprise si symptomatique
- Rétraction radiculaire excessive → symptômes transitoires en général

### 17.2.4 Complications thrombo-emboliques

| Complication | Fréquence | Prévention |
|-------------|-----------|-----------|
| DVT profonde | 10–30 % (Dearborn et al., 1999, *Spine*) (sans prophylaxie) | HBPM + bas de contention + lever précoce |
| Embolie pulmonaire | 0,5–1 % | Idem |
| TEP mortelle | 0,1 % | Idem |

**HBPM** (enoxaparine, tinzaparine) : début à H12–24 post-op si hémostase correcte.

## 17.3 Complications à moyen terme (1–12 mois)

### 17.3.1 Pseudarthrose

**Définition** : Absence de fusion osseuse solide à 12–24 mois → mobilité résiduelle des implants → fatigue → rupture de tige.

**Facteurs de risque** :
- Tabagisme (× 2–3)
- Ostéoporose
- Dénutrition / insuffisance vitamine D
- Greffe insuffisante
- Infection sur matériel (biofilm)
- Correction excessive (stress trop élevé sur les implants)

**Diagnostic** :
- Radio dynamique (flex/ext) → mobilité entre two niveaux
- Scanner (sans injection) → visualisation travées osseuses

**Traitement** :
- Révision avec nouvelle greffe + BMP-2 si adulte + instruments plus rigides
- Ablation matériel si infection associée

### 17.3.2 Rupture de tige

- **Conséquence** de la pseudarthrose le plus souvent
- Parfois fatigue de matériel (implant défectueux)
- Révision si douleur ou perte de correction

## 17.4 Complications tardives (> 1 an)

### 17.4.1 Adjacent Level Disease (ALD)

- **Définition** : Dégénérescence accélérée des disques et articulations adjacents à la fusion
- **Prévalence** : 20–30 % à 10 ans (symptomatique : 10–15 %)
- **Mécanisme** : Augmentation des contraintes de flexion aux niveaux limitrophes
- **Traitement** : Extension de la fusion si invalidant

## 17.5 Cas clinique — Sophie (suite)

> **Données** : J+3 mois, fièvre 39°C, CRP 280, douleur dorsale intense, cicatrice normale.

**Diagnostic clinique** : Infection profonde du site opératoire post-scoliose probable

**Conduite** :
1. Bilan bio urgent : NFS, CRP, hémocultures
2. **IRM rachis** → confirme l'infection (hypersignal T2 périimplant, abcès)
3. Reprise chirurgicale en urgence : **DLMM** (débridement-lavage-maintien du matériel)
4. 5 biopsies peropératoires pour cultures (dont anaérobies)
5. Antibiothérapie IV probabiliste (céfazoline + rifampicine si suspicion staph) → adaptation à l'antibiogramme
6. Durée totale antibiothérapie : 6 semaines IV + 6 semaines orale

---

## Points clés à retenir

> 📌 **1.** L'**hématome épidural** est l'urgence postopératoire absolue — reprise dans les 6–8h pour préserver la fonction neurologique.

> 📌 **2.** L'infection profonde précoce (< 3 mois) se traite par **DLMM** (maintien du matériel) + antibiothérapie prolongée.

> 📌 **3.** La **pseudarthrose** est la principale cause de rupture de tige — le tabac est son facteur de risque majeur.

> 📌 **4.** La **vancomycine topique** (poudre intraopératoire) réduit les infections profondes après arthrodèse.

> 📌 **5.** L'**ALD** est inévitable à long terme — limiter la fusion au minimum nécessaire.
