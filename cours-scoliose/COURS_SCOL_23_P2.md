## 23.7 Traumatismes rachidiens cervicaux hauts (C1-C2)

### Fracture de Jefferson (C1 — burst fracture)

**Mécanisme** : compression axiale pure (plongeon, chute sur la tête).

**Description** : fracture en 4 points de l'atlas (anneau de C1) → élargissement bilatéral des masses latérales.

**Classification de Landells** :
| Type | Description |
|------|------------|
| I | Arc antérieur uniquement |
| II | Arc postérieur uniquement |
| **III** | **Les deux arcs (fracture classique de Jefferson)** |

**Stabilité** : dépend du ligament transverse de l'atlas.
- **Test de Spence** (radiographie de face bouche ouverte) : si déplacement latéral cumulé des masses > **7 mm** → rupture du ligament transverse probable → instabilité.

**Traitement** :
- Stable (ligament intact) : collier cervical rigide 3 mois
- Instable (ligament rompu) : halo-veste ou chirurgie (fusion C1-C2 ou occipito-C2)

### Fracture de l'odontoïde (C2)

**Classification d'Anderson et d'Alonzo** (Anderson & D'Alonzo, 1974, *J Bone Joint Surg Am*) :

| Type | Description | Traitement |
|------|------------|-----------|
| **I** | Apical (au-dessus de la base) | Rare, stable — collier cervical |
| **II** | **Base de la dent (la plus fréquente)** | **Chirurgie recommandée** (pseudarthrose 30–40 % avec ortèse seule) |
| III | Corps de C2 (fracture corporéale) | Collier cervical ou halo 3 mois |

**Traitement du type II** :
- **Vissage antérieur de la dent** (1 ou 2 vis) si fracture oblique favorable (pas de comminution)
- **Fusion C1-C2 postérieure** (technique Harms) si fracture comminutive ou ostéoporose

### Fracture du Pendu (Hangman — C2 bilatérale)

**Mécanisme** : hyperextension + distraction (pendaison, AVP — freinage brusque tête en avant).

**Lésion** : fracture bilatérale de l'isthme C2 (pars interarticularis) + parfois antélisthésis C2/C3.

**Classification de Levine-Edwards** :
| Type | Déplacement | Traitement |
|------|-----------|-----------|
| I | < 3 mm, aligné | Collier cervical 3 mois |
| II | Déplacement + angulation | Halo-veste ou chirurgie |
| IIa | Angulation sans déplacement | Halo-veste |
| III | Déplacement + luxation facettaire | Chirurgie (fusion C2-C3) |

## 23.8 Classification AO Spine — Détail Thoraco-lombaire

### Types A — Compression

| Sous-type | Description | Traitement |
|-----------|------------|-----------|
| A0 | Fracture mineure (apophyse, processus transverse) | Antalgiques |
| A1 | Tassement cunéiforme ≤ 20° | Corset |
| A2 | Fracture sagittale (split) | Corset ou chirurgie si > 30° |
| **A3** | **Burst fracture partielle (1 plateau)** | Corset si stable (variable) |
| **A4** | **Burst fracture complète (2 plateaux)** | Chirurgie si déficit ou > 50 % canal |

### Types B — Tension Band (distraction postérieure)

| Sous-type | Description | Instabilité |
|-----------|------------|------------|
| B1 | Chance osseuse (fracture horizontale post.) | Modérée — chirurgie si > 20° |
| B2 | Flexion-distraction + rupture ligamentaire | Instable → chirurgie |
| B3 | Hyperextension-distraction | Instable → chirurgie |

### Type C — Translational

**Très instable** : déplacement > 3 mm dans n'importe quel plan → TOUJOURS chirurgical.

## 23.9 Score TLICS — Décision chirurgicale

Le **TLICS** (Thoracolumbar Injury Classification and Severity score, Vaccaro 2005) est un outil décisionnel :

| Paramètre | Score |
|-----------|-------|
| **Morphologie** : compression = 1, burst = 2, translational/rotationnel = 3, distraction = 4 | 1–4 |
| **Complexe ligamentaire postérieur** : intact = 0, suspect = 2, rompu = 3 | 0–3 |
| **Statut neurologique** : intact = 0, racine = 2, incomplet = 3, complet = 2, SQC = 3 | 0–3 |

**Décision** :
- Score **≤ 3** → traitement conservateur
- Score **= 4** → équivalent chirurgie/conservateur (décision individuelle)
- Score **≥ 5** → chirurgie recommandée

## 23.10 Neuroprotection médullaire — État de l'art

### Méthylprednisolone — Abandon

La méthylprednisolone haute dose (protocole NASCIS-II) a été **abandonnée** par les guidelines AANS/CNS 2013 :
- Pas de bénéfice neurologique démontré
- Complications significatives : sepsis pulmonaire, infections, hémorragie digestive

### Gestion tensionnelle (standard actuel)

**Objectif TA** après traumatisme médullaire :
- **PAM ≥ 85–90 mmHg** pendant 7 jours (perfusion médullaire) — recommandation AANS 2013
- Vasopresseurs (norépinéphrine) si choc neurogène
- Éviter l'hypotension (aggrave l'ischémie médullaire)

### Chirurgie précoce — Early Surgery

**Evidence** : étude STASCIS (Fehlings 2012, *PLoS ONE*) :
- Chirurgie < 24h vs > 24h
- Amélioration ASIA ≥ 2 grades : **19,8 % vs 8,8 %** (p = 0,02)
- Bénéfice maximal pour les lésions incomplètes (ASIA B-C)

> 🔬 **Expert** : Les thérapies neuroprotectrices en cours d'évaluation (riluzole, minocycline, hypothermie locale) n'ont pas encore de recommandation de grade A. Le standard reste : décompression chirurgicale précoce + maintien de la PAM.

## 23.11 Cas clinique

> 🏥 **Cas clinique** : Thomas, 32 ans, plongeur en piscine, présente une tétraparésie incomplète ASIA C après plongeon dans un bassin peu profond. IRM : fracture-luxation C5-C6 (type C AO Spine) + compression médullaire antérieure. Score TLICS = 8.

**Décision** :
1. **Chirurgie en urgence < 6–8h** (lésion incomplète évolutive)
2. Voie antérieure ACDF C5-C6 + décompression + cage + plaque
3. Si instabilité postérieure associée → fusion complémentaire postérieure
4. PAM maintenue ≥ 85 mmHg en soins intensifs pendant 7 jours
5. Rééducation intensive en centre neuro-médullaire dès J3

## QCM

**Q1.** Dans la fracture de Jefferson (C1), le signe radiologique évocateur d'une rupture du ligament transverse est :
a) Déplacement latéral cumulé des masses > 5 mm
b) Déplacement latéral cumulé des masses > 7 mm
c) Fracture de l'apophyse épineuse C1
d) Signe de la larme sur la radio de profil

<details><summary>💡 Réponse</summary>

**b)** Le test de Spence : si le déplacement latéral cumulé des masses de C1 dépasse 7 mm sur la radio de face bouche ouverte, une rupture du ligament transverse est probable, rendant la fracture instable.
</details>

---

**Q2.** La fracture de l'odontoïde de type II est traitée chirurgicalement car :
a) Elle est toujours instable
b) Le taux de pseudarthrose sous orthèse seule est de 30–40 %
c) Elle comprime toujours la moelle
d) Elle survient exclusivement chez les personnes âgées ostéoporotiques

<details><summary>💡 Réponse</summary>

**b)** La fracture de la base de la dent (type II) a un taux de pseudarthrose de 30–40 % avec traitement orthopédique (halo ou collier), ce qui justifie une fixation chirurgicale (vissage antérieur ou fusion C1-C2 postérieure) dans la majorité des cas.
</details>

---

**Q3.** Selon l'étude STASCIS, la chirurgie < 24h d'une lésion médullaire incomplète (ASIA B-C) donne un taux d'amélioration de ≥ 2 grades ASIA de :
a) 5 %
b) 8,8 %
c) 19,8 %
d) 35 %

<details><summary>💡 Réponse</summary>

**c)** L'étude STASCIS (Fehlings 2012) montre que la chirurgie précoce < 24h double le taux d'amélioration neurologique : 19,8 % d'amélioration ≥ 2 grades ASIA vs 8,8 % pour la chirurgie tardive > 24h (p = 0,02).
</details>
