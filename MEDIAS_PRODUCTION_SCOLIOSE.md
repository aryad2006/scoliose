# CATALOGUE DE PRODUCTION MÉDIAS — FORMATION SCOLIOSE
## Guide complet des images, schémas et animations à produire

### Légende des types de médias

| Icône | Type | Format recommandé | Description |
|-------|------|-------------------|-------------|
| 📷 | **IMAGE** | PNG/JPG haute résolution | Photo clinique, radiographie, capture d'écran |
| 📐 | **SCHÉMA** | SVG/PNG vectoriel | Dessin anatomique, diagramme, illustration |
| 🎬 | **ANIMATION** | MP4/GIF 30-60s | Animation 2D/3D explicative avec mouvement |
| 🎥 | **VIDÉO** | MP4 HD 1080p | Vidéo chirurgicale, démonstration clinique |
| 📊 | **INFOGRAPHIE** | PNG/PDF | Tableau visuel, organigramme, synthèse |
| 🧊 | **MODÈLE 3D** | Interactif (WebGL/Complete Anatomy) | Modèle 3D rotatif et interactif |

### Numérotation : `MXX-SYY-ZZZ`
- M = Module, S = Section, ZZZ = numéro séquentiel du média
- Exemple : `M01-S01-001` = Module 1, Section 1.1, premier média
