14.2 Cyphose posturale

Définition : Hypercyphose thoracique réductible chez un adolescent, sans modifications structurales des vertèbres.

Caractéristique Valeur
Âge d'apparition 10–15 ans
Cobb typique 45–55°
Réductibilité Complète en hyperextension
Radio Normale (pas de cunéiformisation)
Évolution Régressif avec traitement physique

Traitement :
- Rééducation posturale intensive (renforcement paravertébral, étirements pectoraux)
- La cyphose posturale répond très bien à la rééducation
- Pas d'indication chirurgicale
- Corset (Cervico-thoracique) si > 55° et persistance malgré rééducation (rare)

14.3.1 Définition et critères diagnostiques

Critères de Sorensen (radio) :
- Cunéiformisation vertébrale ≥ 5° sur ≥ 3 vertèbres consécutives (critère principal)
- Irrégularités des plateaux vertébraux
- Nœuds de Schmorl (hernie intradiscale à travers les plateaux)
- Disques étroits
- Courbure thoracique mal réductible en hyperextension

14.3.2 Physiopathologie

  • Mécanisme : Ostéochondrose de croissance des plateaux vertébraux → cartilage de croissance vulnérable → hyperpression antérieure → cunéiformisation progressive
  • Facteurs favorisants : charges axiales répétées pendant la croissance (agriculture, sports à impact), prédisposition génétique
  • Chronologie : Typiquement entre 12 et 16 ans (pic de croissance)

14.3.3 Formes cliniques

Forme Apex Prévalence Symptômes
Thoracique T7-T9 75 % Douleur dorsale + déformation
Thoracolombaire T10-T12 20 % Douleur lombaire haute, soulèvement d'objets douloureux
Lombaire L1-L3 5 % Douleur lombaire, peu de déformation

14.3.4 Prise en charge selon la sévérité

Cobb Risser Traitement
< 45° Tout âge Rééducation (étirements pectoraux + renforcement dorsal)
45–70° 0–2 (croissance) Corset Milwaukee ou corset d'hyperextension (23h/j) + rééducation
45–70° 4–5 (maturité) Rééducation seule — chirurgie si douleur réfractaire
> 70–75° Tout âge Discussion chirurgicale si douleur + échec conservateur

Corset de Scheuermann :
- Corset d'hyperextension (Milwaukee, TLSO Méhon d'hyperextension)
- Principe : force de correction en extension sur l'apex cyphotique
- Résultat modéré (~20–30 % de correction)

14.3.5 Chirurgie de la cyphose de Scheuermann

Indications :
- Cobb ≥ 70–75° + douleur réfractaire (> 6 mois de traitement conservateur)
- Déficit neurologique (très rare, myélopathie par cyphose aiguë)
- Déformation esthétique majeure inacceptable avec demande forte du patient

Technique :
- Arthrodèse posterieure avec Ponte osteotomies sur 3–5 niveaux pour restaurer la flexibilité
- Niveaux de fusion : T2/T3 → T12 (sélectionner rigoureusement les vertèbres limites)
- Correction cible : ramener la cyphose à 40–50°
- Si très rigide (< 40 % de correction passive) → voie antérieure complémentaire (discectomies libératrices)

Résultats :
- Correction moyenne : 55–65 % du Cobb initial (Bradford et al., 1974 ; Lowe, 1999)
- Satisfaction 85–90 %
- Complications : PJK (30 % si dépassement de la correction — Kim et al., 2008), neurologiques (rare)

14.4 Cyphose congénitale

Types (selon McMaster & Singh, 1999, J Bone Joint Surg Am) :
- Type I (défaut de formation, aplasie ventrale) : progression rapide → myélopathie possible
- Type II (défaut de segmentation, barre antérieure) : progression lente
- Type III (mixte)

⭐ La cyphose congénitale de type I est une URGENCE : risque de myélopathie par compression médullaire antérieure → chirurgie précoce impérative (résection + fusion).

14.5 Cyphose post-traumatique

Mécanisme : Fracture en coin (tassement antérieur) → ciphose angulaire résiduelle

Classification des fractures vertébrales (AO/TLICS — Vaccaro et al., 2005, Spine) :
- Fractures par compression → risque de cyphose résiduelle si > 30–40 %
- Fractures burst → canal intéressé + cyphose potentielle

Traitement :
- Fracture aiguë : vertébroplastie / cyphoplastie si ostéoporotique + douleur
- Cyphose établie et invalidante : ostéotomie + fusion

14.6 Cyphose de l'adulte ostéoporotique

  • Multiple fractures en coin → cyphose progressive → "Dowager's hump" (bosse de la veuve)
  • Prophylaxie : traitement ostéoporose (bisphosphonates, dénosumab, tériparatide) + Ca2+ + vitamine D
  • Cyphoplastie par ballonnet : restauration partielle de la hauteur vertébrale + analgésie → 70 % de succès antalgique

14.7 Rectification de la lordose lombaire

Lordose lombaire efface : disparition de la lordose → dos plat lombaire

Causes :
- Contracture réflexe (lombalgie aiguë, hernie discale)
- Séquelle chirurgicale (flat back, Module 10)
- Spondylarthropathie inflammatoire (SA)

Conséquences :
- Déformation posturale en avant
- Surcharge musculaire → fatigue chronique
- PI-LL mismatch → repercussion équilibre global

14.8 Cas clinique — Nathan (suite)

Nathan, 16 ans, Scheuermann thoracique T4-T12 Cobb 68°, Risser 2, douleur EVA 5/10, rééducation 3 mois sans amélioration.

Analyse :
- Critères de Sorensen : cunéiformisation sur T6-T7-T8 ≥ 5° (confirmée à la radio)
- Type thoracique T4-T12, Cobb 68° → modérément sévère
- Risser 2 → croissance active

Décision :
1. Corset d'hyperextension (type Milwaukee ou Boston hyperextension) à 23h/j
2. Rééducation posturale renforcée (étirements pectoraux, natation, musculation dorsale)
3. Contrôle radio à 4 mois dans le corset
4. Si correction < 15° dans le corset → discuter la chirurgie (Cobb > 70° avec progression probable)

Si Cobb progresse à 75° à 18 mois → fusion T3-T12 (Ponte osteotomies T5-T10) + instrumentation pédiculaire.

Points clés à retenir

📌 1. La cyphose posturale est complètement réductible et répond à la rééducation — ne pas opérer.

📌 2. La maladie de Scheuermann est diagnostiquée par les critères de Sorensen : cunéiformisation ≥ 5° sur ≥ 3 niveaux consécutifs.

📌 3. Le corset dans Scheuermann est indiqué pour les Cobb 45–70° pendant la croissance (Risser 0–2).

📌 4. La chirurgie de la cyphose est discutée à partir de 70–75° avec douleur réfractaire.

📌 5. La cyphose congénitale type I est une urgence chirurgicale (risque myélopathique).

📌 6. La cyphose ostéoporotique multiple → traitement de fond (anti-ostéoporotiques) + cyphoplastie en phase aiguë.
