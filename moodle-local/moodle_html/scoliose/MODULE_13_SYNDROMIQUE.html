13.1 Syndrome de Marfan

13.1.1 Généralités

  • Prévalence : 1/5 000 à 1/10 000
  • Gène : FBN1 (fibrilline-1, chromosome 15) — transmission autosomique dominante
  • Diagnostic : Critères de Ghent révisés 2010 (score systémique + aorte + ectopie cristallin + génétique)

13.1.2 Scoliose dans Marfan

Paramètre Valeur
Prévalence scoliose 60–70 % des Marfan (Sponseller et al., 1995, J Bone Joint Surg Am)
Type de courbe Variable — souvent double, thoracolombaire
Flexibilité Hypermobile — corrigibilité importante mais récidive fréquente
Particularité Dolicho-scoliose : courbures longues, membres longs, thorax étroit

Comportement atypique :
- La flexibilité élevée rend la correction facile mais augmente le risque de pseudarthrose (tissus conjonctifs laxes = consolidation difficile)
- Risque de correction excessif → surveiller l'équilibre
- Résultats moins bons qu'en SIA à long terme (25–30 % de reintervention à 10 ans)

13.1.3 Bilan préopératoire spécifique Marfan

Examen Objectif Seuil d'alerte
Échocardiographie + Z-score aortique Dépister dilatation aorte Z-score > 3 ou > 45 mm = contre-indication relative
IRM médullaire Ectasie durale (1/3 des Marfan), méga-dure-mère Dure-mère amincie → risque de brèche
Consultation ophtalmologique Ectopie cristallin Anesthésie à adapter si mauvaise coopération
Bilan coagulation Fragile Standard

⭐ Règle absolue : Ne jamais opérer la scoliose si l'aorte est > 45 mm et n'est pas opérée — risque de dissection aortique peropératoire lethal.

13.1.4 Traitement médical associé

  • β-bloquants (aténolol, carvédilol) : réduisent la progression de la dilatation aortique
  • Losartan : alternative/complément (antagoniste TGF-β)
  • Suivi cardiologique annuel

13.2 Neurofibromatose de type 1 (NF1)

13.2.1 Généralités

  • Prévalence : 1/3 000 (Friedman, 1999) — la plus fréquente des phacomatoses
  • Gène : NF1 (neurofibromine, chromosome 17q) — autosomique dominant
  • Critères diagnostiques : ≥ 2 parmi : taches café-au-lait > 5 mm × 6, éphélides axillaires/inguinales, neurofibromes (≥ 2 ou 1 plexiforme), gliome optique, nodules de Lisch, lésion osseuse distinctive, parent NF1

13.2.2 Scolioses dans NF1 : deux types

Type Caractéristiques Pronostic
Non-dystrophique Identique SIA, long, flexible Bon, traitement comme SIA
Dystrophique Courte (4–6 niveaux), angulaire, rigide, foramens élargis, érosions côtes en crayon, vertèbres en coins, neurofibromes intrarachidiens Grave — progression rapide, risque neurologique

Diagnostic du type dystrophique : critères de Crawford :
- ≥ 4 des signes suivants : scalloping vertébral, foramens élargis, érosion costale en crayon, côte ruban, angulation vertébrale, vertèbre en coin

13.2.3 Prise en charge

Dystrophique :
- IRM systématique (neurofibromes intrarachidiens fréquents)
- Chirurgie précoce (< 25°) si type dystrophique confirmé
- Fusion combinée antérieure + postérieure (risque élevé de pseudarthrose si postérieure seule)
- Ne pas retarder — les neurofibromes peuvent évoluer rapidement

Non-dystrophique :
- Même traitement que SIA

13.3 Syndrome d'Ehlers-Danlos (SED)

  • Type : SED hypermobile (hEDS) le plus souvent
  • Prévalence scoliose : 30–50 % des hEDS
  • Comportement : Courbures flexibles, correction facile mais pseudarthrose fréquente (collagène déficient)
  • Chirurgie : Eviter si possible, correction conservatrice maximale ; si chirurgie = renforcement de l'ancrage, utilisation de croix multiples

13.4 Ostéogenèse imparfaite (OI)

Classification de Sillence (Sillence et al., 1979, J Med Genet)

Type Sévérité Scoliose
I Légère 20–30 %
II Létale périnatale —
III Sévère Quasi-universelle (> 80 %) (Engelbert et al., 1998)
IV Modérée 30–50 %
V–XII Variantes Variable

Prise en charge de la scoliose dans l'OI

  • Bisphosphonates (acide zolédronique IV) : réduisent les fractures, augmentent la densité osseuse → réduisent la progression de la scoliose chez certains patients
  • Chirurgie :
  • Risque hémorragique majeur (fragilité vasculaire)
  • Vis pédiculaires difficiles (os mou) → augmentation pédiculaire ± tiges téléscopiques
  • Anesthésie particulière : hyperthermie maligne rapportée, dents fragiles (laryngoscopie prudente)

13.5 Achondroplasie

  • Gène : FGFR3 (gain de fonction)
  • Scoliose présente dans 10–20 % des achondroplasies
  • Souvent associée à sténose canalaire (canal trop étroit) → symptômes neurologiques
  • Risque de compression foraminale sévère
  • Chirurgie : décompression souvent prioritaire sur la correction de la scoliose

13.6 Prader-Willi, Angelman, Rett

Syndrome Scoliose Particularité
Prader-Willi 60–80 % (Holm et al., 1993) Obésité complique l'anesthésie + difficulté de correction
Rett 80–90 % (Kerr et al., 2003) Progressive, longue, commence jeune — souvent chirurgie
Angelman 15–30 % Variable

13.7 Approche multidisciplinaire

Pour toute scoliose syndromique, la prise en charge implique :

Spécialiste Rôle
Chirurgien rachidien Chirurgie, planification
Cardiologue (Marfan, SED) Aorte, dysrythmies
Pneumologue (OI, Prader-Willi) CVF, VNI postop
Généticien Conseil génétique, diagnostic, suivi
Anesthésiste spécialisé Adaptation anesthésie (Marfan, OI)
Rééducateur Pré et post-chirurgie

Points clés à retenir

📌 1. Les scolioses syndromiques doivent être reconnues car elles nécessitent un bilan systémique spécifique.

📌 2. Dans le Marfan : contrôle cardiaque (aorte) obligatoire avant toute chirurgie ; la pseudarthrose est un risque majeur.

📌 3. La NF1 dystrophique est une urgence chirurgicale relative : progression rapide, fusion combinée obligatoire.

📌 4. Dans l'OI : bisphosphonates preventifs, anesthésie spécialisée, ancrage difficile.

📌 5. Ces scolioses s'inscrivent toujours dans une prise en charge multidisciplinaire impliquant généticien, cardiologue et anesthésiste spécialisé.
