## 📝 REFERENCES INTEGRANT MÉDIAS

### Standard de tagging [MEDIA: ...]

Chaque section doit inclure des tags [MEDIA] standardisés :

```markdown
[MEDIA: 📷 PT-MOD05-001 — Radiographie latérale tibia normal, angle PTS 7°]
[MEDIA: 📊 PT-MOD05-002 — Graphique forces biomécaniques Maquet (normal vs élevé)]
[MEDIA: 📐 PT-MOD05-003 — Schéma anatomique plateau tibial sagittal]
```

### Intégration dans CDC files

Chaque fichier CDC_PT_XX reçoit :
- Minimum 3-5 schémas SVG principaux (déjà créés)
- 5-8 radiographies sourçées par section
- 2-3 illustrations anatomiques LCA/PTG

### Intégration dans CAS_CLINIQUES

Chaque cas clinique reçoit :
- 2-4 radiographies du patient (imagerie décrite)
- 1 schéma anatomique pertinent (SVG référence)
- 1-2 algorithmes décisionnels visuels (lien to relevant flowcharts)

### Intégration dans QUIZ

Chaque question image reçoit :
- Fichier radiographique réel ou schéma SVG annoté
- Description légende LMS-compatible
- Points de diagnostic clés marqués
